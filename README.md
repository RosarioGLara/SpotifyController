# 🎵 Spotify Gesture Controller
 
Control Spotify using hand gestures detected through your webcam. Built with MediaPipe, OpenCV, scikit-learn, and the Spotify Web API.
 
---
 
## 📋 Requirements
 
- Python 3.10
- A Spotify Premium account
- A Spotify Developer app ([create one here](https://developer.spotify.com/dashboard))
- Webcam
 
---
 
## ⚙️ Installation
 
### 🍎 macOS (Apple Silicon)
 
**1. Clone the repository**
 
```bash
git clone [https://github.com/RosarioGLara/SpotifyController.git]
cd SpotifyController
```
 
**2. Create a virtual environment**
 
```bash
/opt/homebrew/bin/python3.10 -m venv myenv
```
 
**3. Activate the virtual environment**
 
```bash
source myenv/bin/activate
```
 
You should see `(myenv)` at the start of your terminal line.
 
**4. Upgrade pip**
 
```bash
pip install --upgrade pip
```
 
**5. Install dependencies in this exact order**
 
```bash
pip install "numpy>=1.26,<2.0"
pip install "protobuf>=3.20.3,<4"
pip install "mediapipe==0.10.7"
pip install "opencv-python==4.8.1.78"
pip install "scikit-learn==1.3.2"
pip install joblib spotipy python-dotenv
```
 
**6. Verify the installation**
 
```bash
python -c "
import numpy as np
import cv2
import mediapipe as mp
import sklearn
print('numpy:', np.__version__)
print('cv2:', cv2.__version__)
print('mediapipe:', mp.__version__)
print('sklearn:', sklearn.__version__)
print('All imports OK!')
"
```

## 🔑 Spotify API Setup
 
**1. Create a `.env` file in the project root**
 
macOS/Linux:
```bash
touch .env
```
 
Windows:
```bash
type nul > .env
```
 
**2. Add your Spotify credentials to `.env`**
 
```env
SPOTIPY_CLIENT_ID=your_client_id_here
SPOTIPY_CLIENT_SECRET=your_client_secret_here
SPOTIPY_REDIRECT_URI=http://127.0.0.1:8888/callback
```
 
You can find your credentials in your [Spotify Developer Dashboard](https://developer.spotify.com/dashboard).
 
> ⚠️ Never commit your `.env` file to GitHub. Make sure it is in your `.gitignore`.
 
**3. Add `.env` to `.gitignore`**
 
macOS/Linux:
```bash
echo ".env" >> .gitignore
```
 
Windows:
```bash
echo .env >> .gitignore
```

## 🚀 Usage
 
### Step 1 — Collect gesture data
 
```bash
python collect_dataset.py
```
 
| Key | Gesture |
|-----|---------|
| `p` | Play |
| `a` | Pause |
| `u` | Volume Up |
| `d` | Volume Down |
| `n` | Next Song |
| `r` | Start recording countdown |
| `ESC` | Quit |
 
Press a gesture key first to select the label, then press `r` to start the 3-second countdown before recording begins. Each gesture records for 7 seconds. Repeat for each gesture.
 
### Step 2 — Train the model
 
```bash
python model.py
```
 
This will generate three files inside the `models/` folder:
 
```
models/gesture_model.pkl
models/scaler.pkl
models/encoder.pkl
```
 
> ⚠️ You must retrain the model if you ever change your numpy or scikit-learn version.
 
### Step 3 — Run gesture control
 
Make sure Spotify is **open and actively playing** on your device first, then:
 
```bash
python test_model.py
```
 
Point your hand at the webcam and perform a gesture — it will control Spotify automatically.

## 🛠️ Troubleshooting
 
**`No active device found` Spotify error**
```
Player command failed: No active device found
```
Open Spotify on your device and start playing a song before running `test_model.py`.
 
---
 
**`numpy.dtype size changed` error**
```
ValueError: numpy.dtype size changed, may indicate binary incompatibility
```
Your `.pkl` files were saved with a different numpy version. Retrain the model:
```bash
python model.py
```
 
---
 
**`numpy.core.multiarray failed to import` error**
```
ImportError: numpy.core.multiarray failed to import
```
Your opencv was compiled against a different numpy. Reinstall both:
```bash
pip uninstall numpy opencv-python -y
pip install "numpy>=1.26,<2.0"
pip install "opencv-python==4.8.1.78"
```
 
---
 
**`mediapipe` crashes on Apple Silicon**
```
RuntimeError: ValidatedGraphConfig Initialization failed
```
Use mediapipe 0.10.7 specifically:
```bash
pip uninstall mediapipe -y
pip install "mediapipe==0.10.7"
```
 
---
 
**VSCode shows unresolved imports**
 
Open VSCode from the terminal with the venv already active:
 
macOS:
```bash
source myenv/bin/activate
code .
```
 
Windows:
```bash
myenv\Scripts\activate
code .
```
 
Then select the interpreter: `Cmd/Ctrl + Shift + P` → `Python: Select Interpreter` → choose `myenv`.
